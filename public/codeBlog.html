A Personal Website Using VanillaJS

I've built applications using a very wide variety of frameworking technologies from Spring to Angular. But with my personal site I have a few specific goals in mind that I'd like to accomplish using just as much plain JavaScript as I can.

  1. Light weight. I'd like as little overhead as possible.
  2. Learn responsive design outside of framework

The first goal is established immediately, I have little to worry about in terms of boiler plate code, various framework versions and other issues. The second goal is much more interesting to me as I learned web application design from the framework side. At Fullstack we did some minor projects creating sites from scratch but I wanted to take a deeper dive into understanding and building a responsive web application using only Vanilla JavaScript. I believe that understanding the fundamental behvariors of JS will help my use of the powerful frameworks.
